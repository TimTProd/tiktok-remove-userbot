# TikTok Download Userbot

Telegram-юзербот, который автоматически скачивает видео с TikTok из ссылок в личных чатах и отправляет их в ответ.

## Описание

Этот бот работает как Telegram userbot и отслеживает все входящие личные сообщения. При обнаружении ссылки на TikTok (vm.tiktok.com, tiktok.com и т.д.), бот:
1. Скачивает видео с TikTok
2. Отправляет видео в ответ на исходное сообщение с указанной подписью (если задана)

## Требования

- Python 3.12 или выше
- Telegram API ID и API Hash (получить можно на https://my.telegram.org/apps)

## Установка

1. Клонируйте репозиторий или скачайте файлы проекта

2. Установите зависимости:
```bash
uv sync
```
или
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` в корне проекта со следующим содержимым:
```
API_ID=ваш_api_id
API_HASH=ваш_api_hash
SESSION_NAME=my_userbot
VIDEO_CAPTION=Ваш текст подписи к видео
```

### Параметры конфигурации

- `API_ID` (обязательно) - ваш Telegram API ID
- `API_HASH` (обязательно) - ваш Telegram API Hash
- `SESSION_NAME` (опционально) - имя сессии, по умолчанию `my_userbot`
- `VIDEO_CAPTION` (опционально) - текст подписи, который будет добавлен к отправляемому видео. Если не указан, видео отправляется без подписи.

## Использование

1. uv sync venv

2. uv run main.py

## Зависимости

- `pyrogram` - библиотека для работы с Telegram API
- `python-dotenv` - для загрузки переменных окружения
- `tgcrypto` - для ускорения криптографических операций
- `yt-dlp` - для скачивания видео с TikTok и других платформ
