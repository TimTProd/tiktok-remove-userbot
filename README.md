# TikTok Remove Userbot

Telegram-юзербот, который автоматически удаляет сообщения со ссылками на TikTok в личных чатах и заменяет их на указанный текст.

## Описание

Этот бот работает как Telegram userbot и отслеживает все входящие личные сообщения. При обнаружении ссылки на TikTok (vm.tiktok.com, tiktok.com и т.д.), бот:
1. Удаляет сообщение со ссылкой
2. Отправляет вместо него указанный текст-замену

## Требования

- Python 3.12 или выше
- Telegram API ID и API Hash (получить можно на https://my.telegram.org/apps)

## Установка

1. Клонируйте репозиторий или скачайте файлы проекта

2. Установите зависимости:
```bash
uv sync
```
или
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` в корне проекта со следующим содержимым:
```
API_ID=ваш_api_id
API_HASH=ваш_api_hash
SESSION_NAME=my_userbot
REPLACE_TEXT=Тут была ссылка на TikTok, но она была удалена.
```

### Параметры конфигурации

- `API_ID` (обязательно) - ваш Telegram API ID
- `API_HASH` (обязательно) - ваш Telegram API Hash
- `SESSION_NAME` (опционально) - имя сессии, по умолчанию `my_userbot`
- `REPLACE_TEXT` (опционально) - текст, который будет отправлен вместо удаленного сообщения. По умолчанию: "Тут была ссылка на TikTok, но она была удалена."

## Использование

1. uv sync venv

2. uv run main.py
